INTRODUCTION OF PROJECT


This project is using Python and MongoDB, with a dataset published by Canada Energy Regulator.

The purpose of this project is to practice the following:
  • MongoDB as non-relational database and CRUD with MongoDB
    • Read all records
    • Create new records
    • Update one record
    • Delete one record
    • Search record from database
  • Python programming language, including MVC, Inheritance, Dataframe, Plotly
  • Read CSV file with Python and insert data into MongoDB
  • Data Visualization with Plotly
  • To Build a web application with Flask Framework
According to the dataset provider, the data "This data presents details of pipeline incidents, as defined by the Onshore Pipeline Regulations and Processing Plant regulations, including cause information. The data ranges from 2008 to current; it is updated quarterly."
